# Senhorita Pacman

O projeto _Senhorita Pacman_ foi um projeto de aprendizado de máquina por reforço lecionado por [Daviel Lopes Martins](https://sigaa.ufrn.br/sigaa/public/docente/portal.jsf?siape=1323598) na Universidade Federal do Rio Grande do Norte. O projeto consistia em criar um agente inteligente para jogar o jogo **Ms. Pacman**, utilizando técnicas de aprendizado por reforço com Tensorflow e Gymnasium.

## Ferramentas Usadas

- **Python**
- **Tensorflow**
- **Gymnasium**

## Instalação

Deixei de uma forma bem simplificada, apenas executando esses dois comandos! :)

1. Clone o repositório:
   ```bash
   git clone https://github.com/felipe-sbm/senhorita-pacman
   ```

2. Entre na pasta do projeto:
   ```bash
   cd senhorita-pacman
   ```

3. Instale as dependências:
   ```bash
   pip install --upgrade pip
   
   pip install -r requirements.txt
   ```
